# How it works images

Place the final section images in this folder with these exact filenames:

- positionnement.avif
- maintien.avif
- pelage.avif

Recommended format: portrait images around 466x684 or any 2:3 vertical crop. AVIF is referenced in `index.html`; if you use JPG/PNG/WebP, update the three `src` values in the How it works section.
